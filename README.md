# Web Design Showcase

A collection of responsive web templates built by converting Figma designs into code using HTML, CSS, and SCSS.

This repository contains some of my early frontend practice projects focused on layout building, styling, and responsive design. The main goal was to improve my ability to translate static UI designs into real working webpages.

## Project Type

Frontend design-to-code showcase / early practice projects

## Tech Stack

- HTML
- CSS
- SCSS

## Included Projects

- `landing-page-creatic`
- `portfolio-cards`
- `portfolio-figma`

## Project Overview

These projects were created during my early frontend learning phase as I practiced turning design concepts into responsive web layouts.

The focus was on:

- recreating Figma-based layouts in code
- improving HTML structure
- writing CSS and SCSS styling manually
- understanding spacing, sections, alignment, and visual hierarchy
- building responsive pages for different screen sizes

## What I Practiced

Through these templates I practiced:

- converting design mockups into code
- responsive layout building
- working with HTML and CSS fundamentals
- styling with SCSS
- improving attention to visual structure and UI details

## Why This Repository Matters

This repository represents an important step in my frontend journey because it shows the stage where I was building layouts manually and strengthening core coding fundamentals.

Before focusing more deeply on React, Next.js, and TypeScript, projects like these helped me understand how interfaces are structured and styled from the ground up.
